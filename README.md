Spotify Clone 

A front-end clone of Spotify created using HTML and CSS.

 Features
- Spotify-like landing page
- Fully responsive layout
- Custom scrollbars and hover effects

Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
